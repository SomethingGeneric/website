# Notes on Processors

* Execute one instruction per clock cycle
* Clock cycle speed measured in hertz.
  * Prefixes needed to describe how many cycles happen in a second
* Clock cycle steps:
  * Fetch
  * Decode
  * Execute
* Memory Layers
  * (Internal registers)
  * L1 Cache (in package)
  * L2 Cache (in package)
  * L3 Cache (on motherboard {usually})
  * RAM (sometimes called L4)
  * Hard disk / SSD
